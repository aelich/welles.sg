Beta Version Release v.0.2.0

Released by Welles on 27th July 2019.

Description: In this release we published QuickCheck service with all other pages working some of them are completed in full and some are under construction.

APP Functionalities Pages

  • Home Page - Describes all modes user can access in Welles App

  • Quick Mode Page - Enables user to check is his address tainted or not

  • Flow Mode Page - Enables user to follow flow of address through time. (Under construction)

  • Group Mode Page - Enables user to group up searched address with other addresses owned by owner. (Under Construction)

  • Kyc Mode Page - Enables user to check is owner of searched already determined compliant. (Under Construction)

  • Stat Page - Enables user access to different chats connected to BTC (Only one chart available)

  • Welcome Page - Enables user to see our ForensicBD which entities are dangerous and which are clean

Help, Policy, Terms and More

  • Code-of-Laws - Enable user credible links to laws and regulations in different countries

  • Privacy - Welles Privacy Policy

  • Terms - Welles Terms and Conditions

  • Help - Enables user access to FAQ page

  • Feedback Page - Enables user to get in touch with us

  • Auth Page - Page to control Users Authentication

Details: QuickCheck is enabling to our users feature to test their BTC addresses and to make sure are they clean or are they tainted. Answer is given in form of is valid or invalid, and is followed by color for purpose of intuitive analysis. Our users are checking big number of BTC addresses, to help them to do it in the most effective way, QuickCheck provides a layer of automation so effort can be concentrated on important points.

Technology Used

  • FrontEnd | REACT.JS & D3-visualisation

  • BackEnd 1 | NodeJS & AWS & Docker

  • BackEnd 2 | Haskell, NodeJS, C++, Python3 & AWS & Docker

How to use QuckCheck(Quick Mode)

  • On Maintenance Page Click "Continue" Button

  • On Home Page Click on QuickMode

  • Enter your BTC Address in Input Area

  • Click Search

  • If there is no update click "Refresh" Button

RECAP:

  1. Working QuickCheck algorithm which enables user to test is address clear or tainted.

  2. This version algorithm is connecting to REAL Forensic database(consists of around 25.12 million addresses around ), therefore results are trustworthy.

  3. Functionality we offer in this version is to by entering address of interest and clicking on button "Submit Address" to get is your address "legitimate" or "illegitimate".

  4. Supported on all Browsers.
